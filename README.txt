README

Info:

This app is an inventory manager tool, developed for use as a food inventory manager.  

It was developed by jbrodzinskijr on a XAMPP server stack using a browser for GUI.

Setup:

    • Copy the food/ directory and its contents into the htfiles/ directory on your XAMPP server.  
    • Create a database, such as “food”, where the data will be stored.
    • Import the tables from foodTableStructure.sql into the new database.
    • Adjust the database connection in /includes/food.dbh.inc.php as needed.
    • Access the main page at localhost/food/index.html
